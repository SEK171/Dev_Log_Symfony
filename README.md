# Enterprise Task & Log Management Platform

A containerized full-stack web application built with Symfony (PHP) and Docker for internal task assignment and server log management.
Features role-based user authentication, interactive administrative dashboards, Notifications and secure data ingestion workflows. (to a degree)
The entire architecture is decoupled and containerized;
to run the application locally,
ensure Docker is installed and simply 'docker compose up' to spin up the required web server, PHP environment and database instances.
